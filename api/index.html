invertornot

Make your app's dark mode smarter with invertornot. Our API predicts if an image should be inverted for optimal dark-mode rendering.

Despite dark mode's popularity, images often aren't automatically adapted to fit it, creating a visual mismatch and potential eye strain in low-light conditions. Using our API you can make your website more pleasant to use in dark-mode.

The API works by training an EfficientNet model on a custom dataset. By using deep learning we are able to have a much more reliable approach to solve this problem, the only alternative being heuristics.

API documentations are available here.

The dataset was labeled based on those following CSS filters:

  • invert = 0: filter: grayscale(50%);
  • invert = 1: filter: grayscale(50%) invert(100%) brightness(95%) hue-rotate(180deg);

Usage example:

            
                curl -X 'POST' \
                  'https://invertornot.com/api/url' \
                  -H 'accept: application/json' \
                  -H 'Content-Type: application/json' \
                  -d '[
                  "https://upload.wikimedia.org/wikipedia/commons/e/e3/Queen_Victoria_by_Bassano.jpg"
                ]'
Response: 
            
                [
                  {
                    "invert": 0,
                    "sha1": "da487e2e9855362b4a5f4fdb531c55ae47ac6e1b",
                    "error": "",
                    "url": "https://upload.wikimedia.org/wikipedia/commons/e/e3/Queen_Victoria_by_Bassano.jpg"
                  }
                ]
